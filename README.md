# Otel Rezervasyon Sistemi

Bu proje, web üzerinden otel rezervasyonu yapılabilen bir sistemdir. Masaüstü uygulaması ile de admin ve personel yönetimi vardır.

## Özellikler
- Kullanıcı kaydı ve giriş
- Otel bilgilerini görüntüleme
- Rezervasyon oluşturma ve yönetme
- Web sayfa bilgilerini güncelleme
